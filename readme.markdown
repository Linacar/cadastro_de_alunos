# Cadastro de alunos

### para executar em IDE
Deve ter JDK + eclipse(IDE) instalado
A versão que eu usei do JDK foi 1.8(java 8) mais acredito que se executado em versões mais novas não haverá problema
Você pode importar o projeto no eclipse e rodar a partir da Classe main normalmente como qualquer projeto Java

## Caso queira somente executar o aplicativo compilado
Basta executar o arquivo build/cadastro.jar com o comando 'java -jar caminhoDoArquivo/cadastro.jar'
